# Flask-Blog-Tutorial
A blog application in python using Flask.

# Development

## Create venv

```sh
python -m venv .venv
```

## Activate venv

```sh
. .venv/bin/activate
```

## Install requirements

```sh
pip install -r requirements.txt
```

## Running The App

```bash
python app.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
